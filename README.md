# Novel Game Prototype

ブラウザで動く3D探索型ノベルRPGの試作です。現在は、王都中央広場・ギルド内・教会内を移動しながら、ギルド仮登録までの短い導線を遊べます。

## 現在できること

- タイトル画面から開始
- 小さな3Dの王都中央広場を探索
- ギルド建物に入って、建物内の受付で登録相談
- 教会に入って、身分保証イベントを進行
- アニメ調を意識した低ポリ人型モデルを表示
- 受付・司祭・護衛隊員・古参冒険者などの見た目を差別化
- WASDで移動、矢印キーで視点操作
- 視点方向を基準にした移動
- 1人称 / 3人称カメラ切替
- DualShock 4 / 標準Gamepad API対応
- NPCや入口に近づくと案内を表示
- Eキーで会話・建物出入り
- 会話の選択肢でステータス・目的・クエスト進行を変化

## 操作

### キーボード / マウス

| 操作 | 内容 |
|---|---|
| W / A / S / D | 視点方向を基準に移動 |
| 矢印キー | 視点操作 |
| マウスドラッグ | 視点操作 |
| F | 1人称 / 3人称切替 |
| E | 近くのNPCに話しかける / 建物に入る |
| Enter / Space | 会話を進める |
| Esc | 会話を閉じる |

### DualShock 4

| 操作 | 内容 |
|---|---|
| 左スティック | 移動 |
| 右スティック | 視点操作 |
| × | 会話 / 決定 / 会話送り |
| ○ | 会話を閉じる |
| △ | 1人称 / 3人称切替 |
| OPTIONS | 1人称 / 3人称切替 |

## ファイル構成

```text
.
├── index.html      # 3Dゲーム画面
├── style.css       # HUD・会話ウィンドウ・タイトル画面
├── game.js         # Three.jsによる3D探索処理
├── story.js        # NPC会話・選択肢・クエスト・ステータス
├── THREE_PROTOTYPE.md
└── assets/
    ├── models/     # 将来の3Dモデル置き場
    ├── textures/   # 将来のテクスチャ置き場
    ├── bgm/        # BGM置き場
    └── se/         # 効果音置き場
```

この試作はCDNからThree.jsを読み込みます。インターネット接続がない環境では動きません。DualShock 4はブラウザ・OS・接続方式によりボタン番号がずれる場合があります。
