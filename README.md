# Novel Game Prototype

ブラウザで動く3D探索型ノベルRPGの試作です。現在は、佐藤雄二が森の街道で目覚め、荷車襲撃・商人救助・王都アウレリア入城・ギルド仮登録まで進める導線を遊べます。

## 現在できること

- タイトル画面から開始
- 王都へ続く森の街道から開始
- 森の街道、荷車襲撃現場、王都門を探索
- 黒毛の噛み犬による荷車襲撃イベント
- ファイアボールまたは囮行動で商人を助ける分岐
- 商人紹介状を得て王都ギルド登録へ進む導線
- 王都アウレリア全域マップを探索
- 城壁、北門、中央広場、王城、ギルド、教会、訓練場、市場、貴族街、兵舎、スラム風区画を配置
- 以前より大きい、約420×420スケールの王都マップ
- 外門練習場マップ
- 3Dファイアボールを任意発射
- 1人称視点で照準方向に火球を撃つアクション
- 回避アクション
- デバッグ用の空中浮遊・高速移動モード
- アニメ調を意識した低ポリ人型モデル
- WASDで移動、矢印キーで視点操作
- Shift / R2でスタミナ消費型ダッシュ
- 1人称 / 3人称カメラ切替
- DualShock 4 / 標準Gamepad API対応
- 会話選択肢をDualShock 4で上下選択して決定
- 入口・掲示板などに発光マーカーを配置
- 会話の選択肢でステータス・目的・クエスト進行を変化

## 今回の設計方針

- ファイアボールの場面を王都内ではなく「王都へ続く森の街道」に移動
- 王都は一点マップではなく、城壁・市場・教会・ギルド・王城などの区画を持つ広いマップとして作る
- 広さだけでなく、街道分岐、看板、門、発光マーカーで自然に誘導する
- 「遅さは罪」を意識し、広いマップ用にダッシュとデバッグ高速移動を用意
- なろう系の弱点である“ご都合主義の無双”を避けるため、商人紹介状・身分保証・ギルド審査を進行の根拠にする
- 戦闘操作は今後のために、DualShock 4の□/○/△/R2へアクションを割り振る方向に寄せる
- 巨匠系クリエイターの禁忌として、遅さ・無意味なストレス・論理破綻・スコープ肥大化を避ける方針を維持する

## ファイル構成

```text
├── index.html      # 3Dゲーム画面
├── style.css       # HUD・会話ウィンドウ・タイトル画面
├── game.js         # Three.jsによる3D探索処理
├── story.js        # NPC会話・選択肢・クエスト・ステータス
├── THREE_PROTOTYPE.md
└── assets/
    ├── models/     # 将来の3Dモデル置き場
    ├── textures/   # 将来のテクスチャ置き場
    ├── bgm/        # BGM置き場
    └── se/         # 効果音置き場
```

## 注意

この試作はCDNからThree.jsを読み込みます。インターネット接続がない環境では動きません。DualShock 4はブラウザ・OS・接続方式によりボタン番号がずれる場合があります。
