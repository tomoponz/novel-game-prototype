# Novel Game Prototype

ブラウザで動く3D探索型ノベルRPGの試作です。現在は、佐藤雄二がバナナで死亡して異世界の石畳で目覚め、荷車襲撃・商人救助・紹介状・ギルド仮登録まで進める導線を遊べます。

## 現在できること

- タイトル画面から開始
- 小さな3Dの王都中央広場を探索
- 石畳で目覚めた直後のステータスUI確認イベント
- 黒毛の噛み犬による荷車襲撃イベント
- ファイアボールまたは囮行動で商人を助ける分岐
- 商人紹介状を得てギルド登録へ進む導線
- ギルド建物に入って、建物内の受付で登録相談
- 魔力測定で水晶に細かい亀裂が入るイベント
- 教会に入って、身分保証イベントを進行
- アニメ調を意識した低ポリ人型モデルを表示
- 髪・目・手・靴・装飾などを分けた人型モデル
- 受付・司祭・護衛隊員・古参冒険者などの見た目を差別化
- WASDで移動、矢印キーで視点操作
- 視点方向を基準にした移動
- Shift / R2でスタミナ消費型ダッシュ
- ダッシュ時に視野角を少し広げて疾走感を出す
- 1人称 / 3人称カメラ切替
- DualShock 4 / 標準Gamepad API対応
- 入口・掲示板などに発光マーカーを配置
- NPCや入口に近づくと案内を表示
- Eキーで会話・建物出入り
- 会話の選択肢でステータス・目的・クエスト進行を変化

## 操作

### キーボード / マウス

| 操作 | 内容 |
|---|---|
| W / A / S / D | 視点方向を基準に移動 |
| Shift | ダッシュ |
| 矢印キー | 視点操作 |
| マウスドラッグ | 視点操作 |
| F | 1人称 / 3人称切替 |
| E | 近くのNPCに話しかける / 建物に入る |
| Enter / Space | 会話を進める |
| Esc | 会話を閉じる |

### DualShock 4

| 操作 | 内容 |
|---|---|
| 左スティック | 移動 |
| 右スティック | 視点操作 |
| R2 | ダッシュ |
| × | 会話 / 決定 / 会話送り |
| ○ | 会話を閉じる |
| △ | 1人称 / 3人称切替 |
| OPTIONS | 1人称 / 3人称切替 |

## 今回の設計方針

- 原作シナリオの「バナナ死亡→城下町転移→荷車襲撃→商人紹介状→ギルド登録」をゲーム内の最初の導線にする
- 「チート無双」より先に、信用・紹介状・身分保証という社会制度をゲーム進行にする
- 「遅さは罪」を意識し、通常移動を少し速くし、ダッシュを追加
- ダッシュはスタミナを使うので、速さとリソース消費の小さなリスク/リターンがある
- 長い説明文ではなく、発光マーカーで入口や調べる場所へ自然に誘導する
- 人物モデルは外部3D素材ではなく、低ポリの組み合わせでアニメ調に寄せる

## ファイル構成

```text
.
├── index.html      # 3Dゲーム画面
├── style.css       # HUD・会話ウィンドウ・タイトル画面
├── game.js         # Three.jsによる3D探索処理
├── story.js        # NPC会話・選択肢・クエスト・ステータス
├── THREE_PROTOTYPE.md
└── assets/
    ├── models/     # 将来の3Dモデル置き場
    ├── textures/   # 将来のテクスチャ置き場
    ├── bgm/        # BGM置き場
    └── se/         # 効果音置き場
```
