# 長期ストーリーロードマップ (Long-term Story Roadmap)

関連Issue: #55(長期ロードマップ), #52〜#54。正典の世界設定は `docs/world/kingdom-overworld-plan.md`。

## 0. 主題と構造

舞台は**魔導王国**。主人公は歓迎される英雄ではなく、**制度に捕捉される身元不明者**。
物語の駆動力は戦闘の強さではなく、身分・信用・記録・許可・所属機関、そして
「記録されるほど便利になるが、逃げにくくなる」構造。

魔法学院は王国内の一施設であり、世界の中心ではない（第2章の主舞台ではあるが、王国の一部門）。

各章の書式:
- **舞台** / **主要イベント** / **解放地域** / **明かされること** / **実装優先度**

---

## 第1章 — 王都入城 / ギルド登録 / 学院呼び出し

- 舞台: 森の街道(forestRoad) → 北門 → 王都アウレリア中心区(plaza) → ギルド本部(guildHall) → 外門練習場(trainingGround)
- 主要イベント:
  - 荷車襲撃を火球で退け、商人エドリックを救助（小戦闘 `caravan_combat`）。
  - 商会紹介状を得て北門の一時通行（身元証明ではなく信用の仮置き）。
  - ギルド仮登録・魔力測定（未登録波形 / 測定不能 / ギルドマスター確認）。
  - F級仮登録 → 外門練習場の制御査定 → 模擬戦（危険度判定）→ 学院から呼び出し。
- 解放地域: 中央ハブ(plaza)、ギルド本部、大聖堂記録区、外門練習場、（実績後）魔法学院キャンパス。
- 明かされること: 主人公の詠唱が王国式に該当しない「未登録魔法体系」であること。記録され始める。
- 実装優先度: ★（実装済み。本作の現在地）

## 第2章 — 魔法学院 / 図書館 / 禁書区

- 舞台: 魔法学院キャンパス(academyCampus) / 講義棟(academy) / 学院研究林 / 魔法実験場 / 禁書塔
- 主要イベント:
  - 観察対象としての聴講・詠唱体系の照合。学費と引き換えの「教育と監視」。
  - 図書館での独自詠唱の出自調査 → 禁書区(召喚・危険魔法)への接近。
- 解放地域: 学院研究林、魔法実験場、（上位許可で）禁書塔、魔導技術工房。
- 明かされること: 主人公の魔法が「召喚」系統に近いこと。学院・教会・王権へ情報共有される。
- 実装優先度: M（学院キャンパスは実装済み。研究林/禁書塔は軽量地域マップで段階追加）

## 第3章 — 教会領 / 身元記録 / 召喚疑惑

- 舞台: 教会領・大聖堂都市(churchGrounds) / 教会記録所(church) / 墓地 / 地下記録室
- 主要イベント:
  - 正式な身元記録の手続き。保証人問題。洗礼・出生・移住記録の照合。
  - 墓地・地下記録室で「過去に同種の召喚疑惑があった」記録に触れる。
- 解放地域: 墓地、（高位許可で）地下記録室。
- 明かされること: 主人公の存在が、過去の召喚事件の記録と符合する。教会が警戒を強める。
- 実装優先度: M（churchGrounds 実装済み。墓地/地下記録室は軽量追加）

## 第4章 — 南港市場 / 商会信用 / 取引と依頼

- 舞台: 南港市場 / 商会倉庫街 / 船着き場 / 外国人街
- 主要イベント:
  - 商会信用(Merchant Trust)の蓄積。依頼・取引・密輸の誘い。
  - 無登録者が集まる外国人街での「記録されない生き方」との対比。
- 解放地域: 南港市場、商会倉庫街、外国人街、（信用/許可で）船着き場。
- 明かされること: 王国の管理が及ばない経済圏の存在と、そこからの脱出経路（国外渡航）。
- 実装優先度: M（地域マップを軽量に新設）

## 第5章 — 王城区 / 監視機構 / 行動許可

- 舞台: 王城区（plaza内の上位区画 → 専用内部マップ）
- 主要イベント:
  - これまでの記録（ギルド/学院/教会/商会）が王城区に集約され、行動許可(Crown Record)が発行される。
  - 便利さ（多くの地域が解放）と引き換えに、監視と派遣義務が強まる。
- 解放地域: Crown Record により 辺境砦・山岳街道・国境監視塔・軍駐屯地 などの軍管理地域。
- 明かされること: 主人公は「管理可能な危険」として王国に組み込まれる。逃走の難化。
- 実装優先度: M

## 第6章 — 辺境砦 / 山岳街道 / 王都外への正式派遣

- 舞台: 辺境砦 / 山岳街道 / 国境監視塔 / 軍駐屯地 / 農村 / 商隊路
- 主要イベント:
  - Crown Record を根拠に王都外へ正式派遣。国境・辺境の防衛任務。
  - 派遣の名目で、外縁危険地帯の調査前線に近づけられる。
- 解放地域: 軍管理地域全般、外縁への前哨。
- 明かされること: 王国が外縁の「召喚関連遺構」を主人公に調べさせようとしている意図。
- 実装優先度: L

## 第7章 — 古代遺跡群 / 召喚関連遺構 / 世界外魔法

- 舞台: 古代遺跡群 / 呪われた森 / 魔獣沼地 / 召喚関連遺構（外縁・高危険度）
- 主要イベント:
  - 学院観察+王権許可+教会照会の三者承認による高危険度調査。
  - 主人公の魔法の出自（世界外/召喚）に直接触れる遺構の発見。
- 解放地域: 外縁危険地帯（全照会が前提）。
- 明かされること: 主人公は「世界の外」から召喚された存在であり、王国はそれを利用しようとしている。
- 実装優先度: L

## 第8章以降 — 帰還 / 残留 / 王国に利用されるか拒むか

- 舞台: 召喚関連遺構 / 地下記録室 / 船着き場 / 王城区（分岐）
- 主要イベント（分岐）:
  - 帰還ルート: 召喚を逆向きに使い「世界の外」へ戻る。
  - 残留ルート: 王国に留まり、管理機構の内側から制度を変える / 利用される。
  - 拒絶ルート: 記録から消える＝無登録者として逃走（「消えた者は二度と登録できない」の体現）。
- 解放地域: 章フラグにより最終地域群。
- 明かされること: 「記録されるほど便利になるが、逃げにくくなる」構造の最終的な代償と選択。
- 実装優先度: L

## 実装順の指針

1. **docsと軽量ワールドマップUI**（本PR / 第1〜3章の地域は既存3Dを流用）。
2. 南港市場・辺境砦など優先度Mの地域を**軽量地域マップ**で追加（重い3Dは後回し）。
3. 章進行フラグによる地域解放を `world_map.js` の判定に接続（既に解放条件は `region-access-rules.md` 準拠）。
4. RECORD UPDATED / 許可証台帳 / next_destination との連動を段階追加。
5. 手描き風王国マップ（参考画像のような俯瞰絵）への置換は最終段階。
