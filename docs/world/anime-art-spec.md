# アニメ調ローポリ アートスペック / Anime-Style Low-Poly Art Spec
**対象プロジェクト**: 異世界転生RPG 漆黒の翼 / Novel RPG Prototype（Three.js, 王都アウレリア Aurelia）
**版**: v1.0 — DESIGN NOTE（実装者がそのままビルドできる仕様）

---

## 0. このドキュメントの使い方 / Scope & Philosophy

このドキュメントは **Three.js のプリミティブ（`BoxGeometry` / `CylinderGeometry` / `ConeGeometry` / `SphereGeometry`）だけ**で、外部アセット・テクスチャ・glTF を一切使わずに、明るく清潔な **アニメ／原神（Genshin）風**のキャラと街を組み上げるための具体仕様である。

### 0.1 二軸の両立（最重要・絶対遵守）
| 軸 | 方針 |
|---|---|
| **見た目 VISUAL** | 明るい。彩度高め。空は青。肌は血色。シルエットがアニメ。影は柔らかく、輪郭は読みやすい。Genshin的なファンタジー絵本。 |
| **物語・トーン WRITING** | 地に足のついた社会リアリズム。登録・契約・教会記録・通貨・信用・危険。第一話の「治癒は銀貨だ。安くない」「未契約者は最低限だ」の質感。 |

**両者を同時に持つ。** 絵はかわいく明るいが、世界は制度で動いている。NPCの服の汚れ・階級差・徽章・木札（登録タグ）といった「制度の痕跡」を、**清潔で明るいパレットの上に小物として乗せる**ことで両立させる。汚しは色を濁らせて表現するのではなく、「ベース色＋小さなアクセント片（傷・徽章・包帯・継ぎ当て）」で表現する。

### 0.2 物語の軸（アート小物にも効く）
主人公が集めるのは物理的な「鍵」ではなく、**都市の制度内での信用・身分・居場所**である。したがってアートに登場させるべき「権威の記号」は次のもの：

> ギルド証（首にかける金属タグ）／教会の腕輪（円に三本線）／封蝋付き紹介状／登録された魔力紋／推薦状／通行許可札／保証人の署名。

これらは **NPCやプレイヤーが身につける小道具**として、また **建物の看板アイコン**として繰り返し現れる。「鍵を取りに行く」ダンジョン的ビジュアルは作らない。代わりに「カウンター・台帳・封蝋・徽章・行列」を街の風景に置く。

### 0.3 LUMENメタ設定とのビジュアル接続（控えめに）
世界観の深層では「奇跡＝UI、秩序＝OS」。ゲーム画面上では **ステータス窓・通知（`【NOTICE】You are "Uncontracted"`）が薄い光の輪（光環 / halo ring）として視界端に出る**のが唯一の「UIっぽい」演出。建物やNPCそのものはあくまで実在感のある中世都市として作り、UI光だけがわずかに無機質。これも明るい色で描く（不気味にしない）。

### 0.4 性能の前提（全要素に効く）
- **共有ジオメトリ**：各プリミティブ形状は**1個だけ生成してNPC/建物間で使い回す**（`new BoxGeometry()` を毎フレーム・毎個体で作らない）。
- **共有マテリアル**：色ごとに `MeshLambertMaterial` を1つ作ってキャッシュ（`materialCache[hexColor]`）。`MeshStandardMaterial` は使わない（PBR不要・重い）。アニメ調はランバートで十分、むしろ平坦で良い。
- **インスタンシング**：同種モブ（市場の通行人など）・窓・敷石・屋台は `InstancedMesh` を第一候補に。
- **ライティングは軽量・低**：`HemisphereLight`（空色↔地色）＋ `DirectionalLight` 1灯（影は主要オブジェクトのみ or 無効）。点光源は夜エリアのランタンだけ、本数を絞る。`renderer` は `powerPreference:'low-power'` 相当の軽量設定、`shadowMap` は使うなら `PCFSoftShadowMap` を低解像度で。
- **禁止**：**浮いている建物・浮いているパーツは作らない。** すべての建物は底面が `y=0`（地面）に接地。全パーツは親メッシュにぶら下げ、必ず下の構造に支持される位置に置く。

# PART A — キャラクター / CHARACTERS

## A-1. 「ローポリだが明らかに人」— アニメ頭身の核

ローポリの素体は放っておくと「ただの円柱＋球」になり、人に見えない。**アニメに見せる鍵は頭部の比率とシルエット**。以下を必ず守る。

### A-1.1 基本プロポーション（成人標準・約7頭身の"絵的"圧縮 ≒ 実質5.5〜6頭身）
ユニット：1.0 = 1メートル想定。標準成人 **身長 1.70**。

| 部位 | プリミティブ | 寸法（W×H×D など） | 中心Y | 備考 |
|---|---|---|---|---|
| 頭（頭蓋） | Sphere | 半径 0.16（**やや大きめ**） | 1.52 | アニメ＝頭大きめ。リアル頭身より1.2〜1.3倍 |
| あご（顎を尖らせる） | Cone（伏せ・少し） | 半径0.13・高0.10 | 1.40 | 球の下に逆向き低コーンを差し込み「**やや尖ったあご**」を作る＝アニメ顔の決め手 |
| 首 | Cylinder | 半径0.05・高0.08 | 1.36 | 細く短く |
| 胴（トルソ） | Box（角を意識） | 0.34×0.42×0.20 | 1.12 | 肩を頭幅より少し広く |
| 腰 | Box | 0.30×0.16×0.18 | 0.86 | |
| 上腕 | Cylinder | 半径0.045・高0.24 | 肩から | 左右 |
| 前腕 | Cylinder | 半径0.04・高0.22 | 肘から | わずかに細く |
| 手 | Sphere | 半径0.05 | 手首先 | 簡略化（指は作らない） |
| 脚（腿） | Cylinder | 半径0.06・高0.34 | 0.50 | |
| 脚（脛） | Cylinder | 半径0.05・高0.32 | 0.18 | |
| 足 | Box | 0.10×0.06×0.18 | 0.03 | 前方に長い＝つま先方向が分かる |

> **頭身の意図**：頭半径0.16・身長1.70だと頭が相対的に大きく、目を大きく描けば即アニメ顔になる。**子供NPC**は頭半径0.14のまま身長を1.20に下げる（頭身が下がり、より幼くデフォルメ＝かわいい）。

### A-1.2 顔（BIG EYES）— 平面に「貼る」のではなく薄い板を前に置く
テクスチャ禁止なので、**目・眉・口は極薄のプレートを顔球の前面に配置**して表現する。

| パーツ | プリミティブ | 寸法 | 配置 | 色 |
|---|---|---|---|---|
| 白目台 | Box（極薄 D=0.01） | 0.07×0.05×0.01 | 顔の前面、左右に2枚、わずかにハの字 | ほぼ白 `#FBF7F2` |
| 瞳（**大きく**） | Box or Sphere半球 | 0.05×0.045×0.012 | 白目の上に重ねる | 瞳色（後述パレット） |
| ハイライト | Box極小 | 0.012×0.012×0.013 | 瞳の上1/3に1枚 | 白 `#FFFFFF` |
| 眉 | Box細板 | 0.06×0.012×0.01 | 目の上、角度で表情 | 髪色に近い濃色 |
| 口 | Box細板 | 0.03×0.01×0.01 | あごコーンの少し上 | くすんだ赤 `#C46A5E` |

**ポイント**：瞳を**顔幅に対して大きく**（目の横幅合計が顔幅の約45%）取ると一気にアニメ／Genshin顔になる。ハイライト1枚で「生きた目」になる。眉の角度（水平／への字／つり上げ）だけで受付の事務顔・衛兵の警戒顔・冒険者の不敵顔を作り分けられる。

### A-1.3 髪 — **シルエットを作る3マス構造**（最重要のアニメ化要素）
髪は「球に色を塗る」だけでは絶対にアニメに見えない。**前髪・サイド・後ろ髪の3マスに分け、頭球より一回り大きく張り出させてシルエットを破る**。

| 髪マス | プリミティブ | 役割 | 作り方 |
|---|---|---|---|
| **前髪（bangs）** | Cone（伏せ）×3〜5 を額に扇状 / または Box板を斜めに3枚 | おでこに「房」を作る | 頭球の前上にコーンを下向きで複数刺す。先端で**毛先の三角**＝アニメ前髪 |
| **サイドロック（side-locks）** | Cylinder細 or Box板 を頬の横に左右 | 顔を縦に挟み小顔効果 | 耳位置から下へ垂らす。長さで性別・年齢を演出 |
| **後ろ髪（back-hair mass）** | Sphere（頭球より半径+0.02、後方へ潰す）or Cone大 | 後頭部のボリューム＝横顔シルエット | ポニーテールなら後方にCylinder＋先端Cone |

> **シルエット原則**：真横・真上から見て「頭が真円」だと人形に見える。前髪コーンと後ろ髪で**輪郭にギザつき／張り出し**を作ると、影絵にしても「キャラ」と分かる。これがGenshin的な"立つ"シルエット。

**髪型バリエーション（モブ作り分けの主役）**：
- ショート（前髪短Cone×3＋後ろ髪潰し球）
- ポニーテール（後方Cylinder＋Cone、揺れアニメ任意）
- ボブ（サイドロック長め＋後ろ髪を顎下まで）
- 三つ編み（後方に小Cylinderを2〜3節）
- 坊主／刈り上げ（前髪なし・頭球をそのまま少し暗色）→ 衛兵・荒くれ向き
- フード下（髪マス省略しフードで隠す）→ 教会・スラム

### A-1.4 パレット（明るいアニメ肌・髪・瞳）
すべて**彩度高め・明度高め**。`MeshLambertMaterial` のベース色として使用。日向で白飛びしすぎないよう、原色よりわずかにくすませた"アニメ塗り"の中間色。

**肌（skin）** — 個体差は下記から hash で選ぶ：
| 名称 | hex | 用途 |
|---|---|---|
| 明るい肌 | `#FCE0C2` | 標準・貴族・学生 |
| 標準肌 | `#F2C49B` | 一般市民・冒険者 |
| 日焼け肌 | `#D89B6C` | 衛兵・行商・農夫 |
| 褐色肌 | `#A86B43` | 遠方出身・スラム多様性 |

**瞳（eye）** — アニメは瞳色が鮮やか：
`#3FA9F5`(青) / `#36C275`(緑) / `#9B5DE5`(紫) / `#F2A93B`(琥珀) / `#E5566B`(赤茶) / `#5A6B7A`(灰＝セヴェル系/監察) / `#7A4A2B`(茶)

**髪（hair）** — アニメ色も可だが王都の一般人は自然色多め、職種で寒暖を分ける：
`#3A2A22`(黒茶) / `#6B4A2E`(茶) / `#C9A24B`(金) / `#B23A2E`(赤毛) / `#8FA3B0`(銀灰) / `#2E4A6B`(濃紺＝学生映え) / `#6BBF8A`(若草＝魔法系の遊び)

> 明るく見せるコツ：髪は肌よりやや濃く・服は髪と補色寄りに置くと、ローポリでも顔が引き立つ。

### A-1.5 決定論的バリエーション（モブが全員同じにならない仕組み）
**全モブは index または座席ハッシュから決定論的に外見を振る。** ランダム実行ごとに変わらず、再現性がある（デバッグ・セーブ整合のため）。

```
// 疑似コード。seed = NPCの一意index、または floor(x*73856093 ^ z*19349663)
function pick(arr, seed, salt){ return arr[(hash(seed*31 + salt)) % arr.length]; }

skinColor   = pick(SKIN,  seed, 1);
hairColor   = pick(HAIR,  seed, 2);
eyeColor    = pick(EYE,   seed, 3);
hairStyle   = pick(STYLES,seed, 4);
height      = 1.62 + (hash(seed,5)%17)/100;   // 1.62〜1.78
bodyWidth   = 0.92 + (hash(seed,6)%18)/100;   // 痩せ〜がっしり 0.92〜1.10 スケール
skinTone飾り = (hash(seed,7)%5===0) ? 包帯/傷/そばかす板 : なし; // 5体に1体だけ差し色
```

差し込む**個体差アクセント**（5体に1〜2体だけ付ける＝制度リアリズムの匂い）：
- 包帯（前腕に白Cylinderを薄く巻く）… 第一話の「布で巻かれる」
- 頬の傷（顔に赤茶の極小Box 1枚）… 荷車の男「頬に傷」
- 首の登録タグ（鎖骨に金属Box＋紐Cylinder）… ギルド証
- 教会の腕輪（前腕に布Cylinder・円に三本線の極小印）… 洗礼済みの印
- 背負い袋（背中にBox＋紐）… 行商・冒険者

> これだけで「群衆が個人の集合」に見え、かつ「登録社会・治療費・徽章」という世界観が群衆の中ににじむ。

## A-2. NPCタイプ別 仕様 / Per-NPC-Type Differentiation

各タイプは **(a) シルエット、(b) シード色、(c) 追加プリミティブ片、(d) アクセサリ、(e) 配置区画** で他と一目で区別できること。素体（A-1）は共通、その上に「**職業レイヤー**」を乗せる。

凡例：`色`はベース服色のシード（hash前のデフォルト）。実装では各タイプ専用パレット配列から index で振る。

---

### ① ギルド受付 / Guild Receptionist（例：リサ・フェン）
- **印象**：清潔・事務的・有能。第一話「羽根ペンが早い。手作業で社会を回している」。**つっこみを諦めた受付嬢**の余裕。
- **シルエット**：きっちりした制服。立ち姿勢、書類を持つ手。髪はまとめ髪（後ろ髪を小球で締める／ポニー）。
- **シード色**：制服上衣 `#3E5C76`（落ち着いた青）＋白襟、エプロン/前掛け `#EDE6D8`。
- **追加プリミティブ**：
  - 胸の徽章（金属Box極小 `#C9A24B`、ギルド紋＝槍と盾を示す小三角×2）
  - 名札（胸に白い極薄Box、第一話「胸元の名札」）
  - カウンターに置く羽根ペン（細Cylinder＋先Cone白）と台帳（Box平たい）— NPC固有プロップ
- **アクセサリ**：袖口にインク染み（極小の濃色Box・あれば1個）。
- **配置区画**：冒険者ギルド屋内カウンター。

### ② 冒険者 / Adventurer（例：護衛ガラン・ホルト、模擬戦の相手）
- **印象**：実戦的。革鎧・武器。荒削り。第一話「革鎧。槍。剣」。**見た目悪いがいい人**の枠も含む。
- **シルエット**：肩に装甲ボリューム、腰に武器、ブーツで脚が太く見える。背に武器のラインが立つ。
- **シード色**：革鎧 `#6B4A2E`〜`#8A5A33`、布部 `#556B5D` か `#7A3B3B`、ベルト濃茶。
- **追加プリミティブ**：
  - 胸当て（胴Boxの前に薄Box 1枚、革色）＋肩当て（肩にSphere半球×2）
  - 武器（背に剣＝細長Box＋柄Cylinder ／ または槍＝長Cylinder＋先Cone）
  - ブーツ（脛下を太めCylinderで覆い色を変える）
  - 首に**ギルド証タグ**（必須＝登録者の証）
- **アクセサリ**：頬の傷・包帯（戦闘職なので個体差で高確率）。マント（背にBox板を垂らす、上位ランク表現）。
- **作り分け**：ランクで装備密度を変える（F=布主体／上位=金属多め・マント）。脳筋タイプは bodyWidth を上限、武器を大型化。
- **配置区画**：ギルド前・酒場・依頼掲示板の前。

### ③ 衛兵・門番 / City Guard（例：ブラム・ガーランド）
- **印象**：制式・無表情・規約。第一話「制服のような上着。胸に金属の徽章」「規約どおりに扱う」。
- **シルエット**：そろいの装備。**兜（頭を金属で覆う）でシルエットが角張る**＝市民と一発で違う。槍直立。
- **シード色**：制式の統一色 `#46506B`（紺鋼）＋金属 `#9AA3AD`。**衛兵は色をほぼ振らない**（制式＝統一が記号）。
- **追加プリミティブ**：
  - 兜（頭球の上に Cylinder＋浅Coneのてっぺん、面頬は省略可）
  - 胸甲（胴前に金属色Box、王紋＝王冠の極小Box印）
  - 槍（長Cylinder＋Cone、直立で手に持たせる）／盾（腕前に薄Box円〜角）
  - マント or サッシュ（肩から斜めに布Box板、隊色）
- **アクセサリ**：胸の**王権徽章**（王冠モチーフ）。夜エリアではランタンを持たせる。
- **配置区画**：門・大通り・貴族街境界・役所前。**2体並べて直立**させると"検問"の絵になる。

### ④ 商人・行商 / Merchant（例：エドリック・ヴェイン、市場の売り子）
- **印象**：算盤勘定・身なりは中の上・愛想と打算。第一話「商会の者」「帳簿が身体を動かしている」。
- **シルエット**：ふくらんだ上着、肩掛け鞄 or 背負い荷、布の帽子。やや猫背で手揉み姿勢も可。
- **シード色**：上着 `#7A4A8C`(紫)／`#A8602E`(琥珀茶)／`#2E6B5A`(緑青)など**やや高級色**＋金の縁取り片。
- **追加プリミティブ**：
  - 帽子（頭上に低Cylinder＋つば＝薄い円盤Box、商人帽）
  - 肩掛け鞄（脇にBox＋紐Cylinder）または背負い荷（背中にBox積み）
  - 帯（腰に明色サッシュBox）、指輪相当の金片（手に極小Box・富の表現）
  - 露店なら天秤（小Cylinder＋皿2枚）や金庫箱（Box＋金具）
- **アクセサリ**：**封蝋付き紹介状**（手に持つ巻物＝Cylinder＋赤い封蝋Sphere極小）。商会の証。
- **作り分け**：大商人＝色高級・装飾多／行商＝くすみ色・荷大きめ・帽子つば広。
- **配置区画**：市場の屋台・倉庫・商会・宿屋。

### ⑤ 信徒・シスター / Church Faithful & Sister（例：祈祷会の人々、銀印の少年、監察官セヴェル）
- **印象**：静謐・整然・"柔らかい笑いほど刃が入る"。第一話「黒い外套」「胸に小さな銀の印。円に三本線」。
- **シルエット**：**ローブ＝脚を1本の台形で覆う**（脚を見せない）。フード or ベール。直線的で清潔。
- **シード色**：ローブ `#F4F1EA`(生成り) ／ シスター濃色 `#3A3550` ／ 監察官は黒外套 `#2A2730`。差し色に教会色。
- **追加プリミティブ**：
  - ローブ裾（腰から下を**広がるCone or 台形Box**で覆い、脚プリミティブを隠す＝シルエットが聖職者）
  - フード（頭球の後ろ上にCone／ベールはBox板を後ろに垂らす）
  - 胸の**銀印**（円に三本線＝Cylinder極薄＋上に細Box×3、`#C8CDD2`銀）
  - 監察官は外套を翻す用の背中マント板＋杖（長Cylinder＋上にSphere）
- **アクセサリ**：手に聖典（Box）、香炉（小Cylinder吊り）、洗礼の水盤（屋内プロップ）。
- **作り分け**：一般信徒（生成りローブ・印小）／シスター（濃色ベール）／監察官（黒外套・灰の瞳`#5A6B7A`・杖・無表情眉）。**銀印の少年**は子供スケール＋黒外套＋銀印。
- **配置区画**：大聖堂・教会・祈祷会・路地で監視。

### ⑥ スラム住人・貧民 / Slum Dweller（grounded realism の要）
- **印象**：明るい絵の中の"影"。だが**みじめさを濁色で塗らない**。色は褪せ・継ぎ当て・小柄・うつむき姿勢で表現。第一話「布が荒く、色が暗い」を、明度を落としすぎず"くすませ＋当て布"で。
- **シルエット**：ぼろのマント／フードで顔半分隠れる。背を丸める。裸足 or ぼろ靴。荷を抱える。
- **シード色**：褪せた `#8A8270`(土色)／`#6E6457`(灰茶)／`#5C5A50`。**継ぎ当て＝ベースに別色の小Boxを数枚貼る**（赤茶・青の端切れ）。
- **追加プリミティブ**：
  - フード付きマント（頭〜肩をCone／Box板で覆い、当て布の小Box）
  - 包帯・素足（足Boxを肌色に、または布を巻く）
  - 抱える袋・椀（手に小Box／浅Cylinder＝物乞いの椀）
- **アクセサリ**：**首にタグが無い**ことが記号（未登録＝Uncontracted の視覚化）。痩せ＝bodyWidth下限。
- **配置区画**：路地裏・市壁際・大通りの隅・布で覆われた荷の近く（第一話の路地の死体は直接描かず"覆い布Box＋蝿の点"で示唆＝R15配慮）。

### ⑦ 貴族の使用人 / Noble's Servant（メイド・従者・執事）
- **印象**：清潔・折り目正しい・主人の格を背負う。明るい絵の見せ場（Genshin的にきれい）。
- **シルエット**：制服（メイド＝エプロン台形＋ヘッドドレス／従者＝丈の長い上着）。直立・手を前で組む。
- **シード色**：濃紺 or 黒の上衣 `#2E3140` ＋**白エプロン/白手袋**`#F6F3EC`＋主家の差し色（リボン・サッシュ）。
- **追加プリミティブ**：
  - エプロン（胴〜腰前に白い台形Box）＋ヘッドドレス（頭上に白い小Box＋フリル極小）
  - 白手袋（手Sphereを白に）、襟（首元に白Box）
  - 従者は丈長コート（腰下にBox板を垂らす）＋ベスト
  - 銀盆（手に薄Cylinder＋上にカップBox）など給仕プロップ
- **アクセサリ**：主家の紋章リボン（胸に差し色Box）、**貴族街通行許可札**（腰に下げた小Box＝身分保証の可視化）。
- **配置区画**：貴族屋敷・貴族街・役所への使い・市場での高級買い付け。**汚れゼロ・姿勢良し**が他区画NPCとの差。

### ⑧ 魔法学校生 / Magic Academy Student
- **印象**：若く明るく、制服でそろい、ローブで魔法っぽい。シナリオ後半「学園」の主役層。Genshin学園的なキラ感。
- **シルエット**：**制服ローブ＋杖**。フードは下ろし、髪型が映える（学生は髪で個性）。本を抱える。
- **シード色**：制服ローブ `#2E4A6B`(学院紺) or `#4A2E6B`(紫)＋**学年/寮で差し色のサッシュ**（赤・緑・金・空色）。襟・縁取りは金 `#C9A24B`。
- **追加プリミティブ**：
  - 学院ローブ（肩から羽織る板＋裾、紺）／ベスト＋スラックスの制服
  - **杖**（細Cylinder＋先端にSphere＝魔石、色は寮色／光らせる場合は emissive 弱）
  - 本（脇に抱えるBox）、肩掛け鞄（Box＋紐）
  - 髪は明るめ・遊び色可（`#6BBF8A`若草など）＝学生の華やぎ
- **アクセサリ**：胸の**学院章**（紋章Box）、**登録された魔力紋**を腕に薄く（極小の幾何Box＝魔法使用許可の可視化、第一話の"詠唱が変だ"の対比で"正規の術式印"）。
- **作り分け**：寮色サッシュ＋杖の魔石色を seed で振る。優等生＝姿勢良し本多め／問題児＝着崩し（襟Box傾け）。
- **配置区画**：魔法学校・魔法具店・図書・学生街。**集団で同制服＋個別の髪＝"そろいだが個人"**が映える区画。

# PART B — 建物・街区 / BUILDINGS & DISTRICTS

## B-1. 共通の建築語彙 / Shared Architectural Vocabulary

王都アウレリアは**石造り・木の扉・鉄の金具・布の旗**（第一話）。明るい絵本トーンなので、石は灰ではなく**温かいクリーム石**、屋根は彩度のある瓦色で「街がカラフルだが統一感がある」を作る。

### B-1.1 共通パレット
| 部位 | hex | 備考 |
|---|---|---|
| 壁・石（標準） | `#E8DCC0` | 温かいクリーム石（灰色にしない＝明るさの肝） |
| 壁・漆喰白 | `#F4EEE0` | 上層・教会・貴族 |
| 木組み柱（ハーフティンバー） | `#6B4A33` | 壁面に縦横の梁Boxを走らせると一気に中世＆かわいい |
| 屋根（標準瓦） | `#B5533E`（赤茶） | 街の主調 |
| 屋根（青瓦） | `#3E6B8C` | 公的建物・教会で寒色アクセント |
| 屋根（緑） | `#4E7A55` | 薬屋・魔法系で個性 |
| 扉・窓枠 | `#5A3B26`（濃茶） | |
| 金属看板・金具 | `#9AA3AD` / 金 `#C9A24B` | |
| 地面（敷石） | `#C9BBA0` | InstancedMeshで敷く |

### B-1.2 共通構造ルール（全建物）
1. **接地必須**：底面 `y=0`。**浮遊禁止。** 全パーツは下の構造に支持される位置のみ。
2. **基本形**：壁＝Box、屋根＝Box傾斜 or 三角プリズム（薄Boxを回転 / ConeやWedge代用）、塔＝Cylinder＋Cone。
3. **看板（識別の主役）**：扉の上に**サインボードBox＋アイコン**を必ず付ける。アイコンはプリミティブ数個で作る記号（下記各種）。**この看板アイコンが建物タイプの一発識別**。
4. **扉・窓**：扉＝壁前に濃茶Box（縦長）。窓＝壁にはめる小Box（枠＋明色のガラス面、夜は emissive 弱でランタン色）。窓は `InstancedMesh` で量産。
5. **庇・天幕**：店は扉上に**ストライプの庇**（Box板を斜め、明色×白の2色）＝"お店感"。
6. **旗・幟**：ポールCylinder＋布Box板。区画色・ギルド色で。
7. **性能**：壁/屋根/窓/扉の各形状は共有ジオメトリ。同型建物（長屋・倉庫列）は `InstancedMesh`。装飾の小物（旗・樽・看板）は LOD的に近距離のみ表示で可。

| 貴族街 | 漆喰白＋青瓦 | 清潔・広い・門で仕切る | 貴族屋敷・整形庭園 |

## B-2. 建物タイプ別 仕様 / Per-Building Vocabulary

各建物は **(屋根 / 看板＋サインアイコン / 扉 / 窓 / プロップ / 区画)** で必ず識別できること。第一話準拠の質感（掲示板・台帳・封蝋・薬草の匂い・酒の匂い）を小物で出す。

---

### ① 冒険者ギルド / Adventurer's Guild
> 第一話「石造りの二階建て。入口の上に金属の看板。槍と盾の意匠」。街の入口＝制度の入口。
- **屋根**：青瓦寄りの大きめ切妻（公的）。2階建て、堂々と。
- **看板＋アイコン**：金属看板Box＋**槍と盾の紋章**（細長Box＝槍 ×2 を交差、後ろに角丸Box＝盾、金`#C9A24B`）。
- **扉**：両開きの大扉（濃茶Box×2、金具点）。
- **窓**：1階に依頼で人が見える大窓、2階に整列窓。
- **プロップ**：**外壁に依頼掲示板**（柱2本＋板Box、紙＝白い小Box複数、端が波打つ＝わずかに傾ける）。入口に衛兵/職員NPC、剣の擦れ音の冒険者。屋内＝**カウンター（長Box）＋台帳＋羽根ペン＋魔力測定の水晶（透明Cylinder、ヒビ＝白い細線Box）**。
- **区画**：公共／商業の境、大通り沿い。

### ② 武器屋 / Weapon Shop
- **屋根**：赤茶切妻、こぢんまり2階。
- **看板＋アイコン**：**剣**（長Box＝刃＋短Cylinder＝柄、銀＋金鍔）。または交差した剣2本。
- **扉**：片開き濃茶、上に鉄帯。
- **窓**：ショーウィンドウ大（武器が並ぶ＝中に細Box数本立て）。
- **プロップ**：店先に**武器ラック**（横Cylinder＋立てかけた剣/槍Box数本）、研ぎ石（Cylinder）、砥水桶。看板の脇に小ハンマー記号。
- **区画**：商業区、防具屋の隣に置くと"武具通り"。

### ③ 防具屋 / Armor Shop
- **屋根**：赤茶。武器屋と差別化するため**入口が金属質**。
- **看板＋アイコン**：**盾と兜**（角丸Box＝盾＋上に半球Sphere＝兜）。
- **扉**：鉄補強の重い扉（濃茶Box＋金属帯Box横3本）。
- **窓**：小さめ（防御＝堅牢の表現）、鉄格子（細Box格子）。
- **プロップ**：店先に**鎧立て**（十字の支柱＝Box＋肩甲Sphere＋胸甲Box＝マネキン鎧）、盾の立てかけ、ふいご/金床（Box＋上に黒Box）で鍛冶兼業も可。
- **区画**：武器屋の隣。

### ④ 薬屋・薬師 / Apothecary
> 第一話の治癒院に連なる"薬草の匂い・青い消毒液・黄緑の軟膏"。
- **屋根**：**緑屋根**`#4E7A55`（薬草＝植物の色で一発識別）。
- **看板＋アイコン**：**乳鉢と乳棒**（浅Cylinder＋細Cylinder斜め）／または**薬瓶**（小Cylinder＋コルクSphere、青or黄緑の液色）。
- **扉**：明るい木扉＋上に乾燥ハーブの束（細Cone逆さ＝吊るし草、緑）。
- **窓**：棚が見える窓（中に**色とりどりの小瓶＝小Cylinder列**、青`#A8D8E8`・黄緑`#C8D86A`・赤）。
- **プロップ**：店先に**薬草の鉢**（Cylinder鉢＋上に緑Sphere/Cone）、吊るし草、天秤。治癒院兼業なら**治癒院の印の木札**プロップ。
- **区画**：商業区・ギルド近接（怪我人動線）。

### ⑤ 宿屋 / Inn
> 第一話「木の看板に、曲がった匙の絵」＝"曲がった匙亭"。屋根の価値が高い世界。
- **屋根**：赤茶の大屋根、煙突（Cylinder＋煙Sphere点々）＝"泊まれる・温かい"。3〜2階で客室窓が並ぶ。
- **看板＋アイコン**：**ベッド or 匙**（曲がった匙＝細Cylinderを軽く曲げた風＝先にSphere）。吊り下げ看板（横棒＋鎖Box）。
- **扉**：歓迎の広めの扉、上にランタン。
- **窓**：**各階に等間隔の客室窓**（夜は暖色 emissive＝"灯りがついてる宿"が街の安心記号）。
- **プロップ**：入口にランタン（Cylinder＋黄光）、ベンチ、水桶、看板下に"空室"札（小Box）。
- **区画**：商業区・ギルドから少し離れた通り（第一話どおり）。

### ⑥ 酒場 / Tavern
> 第一話「酒の匂いが濃い。笑い声が荒い」「酒場が黒帳（情報が集まる）」。
- **屋根**：赤茶、低めでどっしり。横に広い。
- **看板＋アイコン**：**ジョッキ／樽**（短Cylinder＝ジョッキ＋取っ手Box ／ 樽＝Cylinder＋帯Box）。泡＝白Sphere点。
- **扉**：両開きの軽い扉（スイングドア風、濃茶Box×2を少し開けて）。
- **窓**：暖色に灯る窓、にぎわいが透ける。
- **プロップ**：**店先に樽（Cylinder複数）・長椅子・テーブル（Box＋脚）**、立て看板（黒板Box＝本日の品）、ランタン。にぎわいNPCを密集配置。
- **区画**：商業区／夜は歓楽街にも。情報屋NPCがここに紛れる。

### ⑦ 魔法具店 / Magic Item Shop
- **屋根**：**紫or濃紺屋根**＋星/月の飾り（小Box）。塔っぽい小尖塔（Cylinder＋Cone）。
- **看板＋アイコン**：**結晶/オーブ**（Sphere＝魔石、emissive弱で淡く光る、青紫）＋星マーク（極小Box放射）。
- **扉**：装飾扉（枠に金縁Box、上にアーチ＝半Cylinder）。
- **窓**：丸窓（Cylinder薄を正面）＋中に光る品（小Sphere emissive）。
- **プロップ**：店先に**杖立て**（細Cylinder＋先Sphere数本）、浮かない範囲の飾り水晶（**必ず台座Box on 地面**＝浮遊禁止）、ルーン看板（壁に幾何Box）。
- **区画**：商業区・学院区寄り（学生客）。

### ⑧ 大聖堂・教会 / Cathedral & Church
> 第一話「石造りで扉がでかい。扉がでかい建物は大抵偉そうだ」。円に三本線の権威。
- **屋根**：**漆喰白の壁＋青瓦＋高い尖塔（Cylinder＋細長Cone）**。バラ窓。最も背が高く街のランドマーク。
- **看板＋アイコン**：建物自体が記号。正面上に**円に三本線の紋章**（Cylinder薄＋細Box×3、銀/金）。尖塔頂にシンボル（小Sphere or 円環Torus代わりにCylinder薄）。
- **扉**：巨大アーチ両開き（高いBox×2＋上に半Cylinderアーチ、金具荘厳）。
- **窓**：**ステンドグラス風＝色違いの小Boxを格子状**（青・赤・金・緑を市松／放射）＝明るい絵の見せ場。**バラ窓**＝正面に大Cylinder薄＋中を色分割。
- **プロップ**：前庭に**洗礼の水盤**（Cylinder台＋浅い水皿）、灯火台、旗（円に三本線）、階段（Box段々）、銀印のNPC。
- **区画**：中央公共区の核。広場に面して正対配置。

### ⑨ 市場の屋台 / Market Stall
> 第一話「露店の干し肉、香辛料、甘い焼き菓子」。明るく雑多で楽しい。
- **屋根**：**ストライプ天幕**（Box板を斜め、明色×白／赤白・青白でカラフル＝市場の華）。柱4本（Cylinder）。
- **看板＋アイコン**：商品そのもの。**台の上に商品プリミティブ**（パン＝Sphere茶／果実＝Sphere赤/黄／干し肉＝Box暗赤吊り／香辛料＝小Cylinderに山Cone色とりどり／布＝畳んだBox）。
- **扉**：なし（オープン）。**カウンター台**（横長Box）。
- **窓**：なし。
- **プロップ**：天秤（Cylinder＋皿2）、籠（Cylinder編み色）、樽、価格札（小Box）、売り子NPC（商人タイプ・帽子）。
- **性能**：**市場は屋台を `InstancedMesh` で量産**し、商品プリミティブだけ seed で種類替え＝"全部同じに見えない市場"。
- **区画**：商業区の広場・通り。密集配置で賑わい。

### ⑩ 倉庫 / Warehouse
- **屋根**：低く大きい片流れ or 緩い切妻、装飾なし（実用）。
- **看板＋アイコン**：**箱/樽スタンプ**（Box＋×印）または商会紋（小Box）。看板は地味。
- **扉**：**大きな搬入扉**（横に広いBox、引き戸風＋金属レール帯）。
- **窓**：少ない・高い位置に小窓（換気）。
- **プロップ**：外に**積み樽・木箱（Box/Cylinderを積層、`InstancedMesh`）**、荷車（Box＋車輪Cylinder×2）、荷運びNPC（Fランク冒険者の仕事＝第三話の"荷運び"）。
- **区画**：商業区の裏・市壁近く・川/門の物流動線。

### ⑪ 貴族屋敷 / Noble Manor
- **屋根**：**漆喰白＋青瓦の複数棟＋角に小塔（Cylinder＋Cone）**、左右対称。最も整い広い。
- **看板＋アイコン**：看板なし＝**門と紋章で示す**。正門（石柱2本＋鉄門Box格子）に**家紋プレート**（Box＋幾何意匠）。
- **扉**：高い両開き＋ポーチ（柱Cylinder＋庇Box＝玄関の張り出し、ただし**柱で必ず支持**＝浮遊禁止）。
- **窓**：**背の高い縦長窓を等間隔**＋バルコニー（手すりBox、床はBoxで接続）。
- **プロップ**：**整形庭園**（生垣＝緑Box列、噴水＝Cylinder台＋水Sphere、敷石の道）、門に衛兵/使用人NPC、**塀でぐるり**（Box塀＋柱）＝貴族街の"仕切り"。
- **区画**：貴族街。塀と門で平民区と分離（通行許可の世界観）。

### ⑫ 役所・記録所 / Registry & Record Office
> 「教会記録・登録・台帳・封蝋」の総本山。物語の"信用と身分"の軸を体現する建物。
- **屋根**：青瓦・左右対称・厳格。正面に**柱列（Cylinder×4〜6＝列柱ポルチコ）**＋三角破風（Wedge/三角Box）＝"お役所＝公的"の記号。
- **看板＋アイコン**：**台帳と封蝋**（開いた本＝Box＋赤い封蝋Sphere）／**羽根ペン**（細Cylinder＋Cone白）。王権＋教会の合意印（円に三本線＋王冠の合体Box）。
- **扉**：重厚な公的扉＋上に紋章。
- **窓**：整然と並ぶ事務窓。
- **プロップ**：**正面に掲示板（門税改定・夜間外出禁止・失踪者情報の札＝白Box複数）**、入口に**順番待ちの行列NPC**＝"制度に並ぶ人々"が役所を一発で説明。階段、衛兵。
- **区画**：中央公共区。ギルド・大聖堂と三角形に配置（王権/教会/ギルドのトライアングルを街割りで暗示）。

### ⑬ 魔法学校 / Magic Academy
> シナリオ後半の主舞台。明るく若く、Genshin的学園。
- **屋根**：**学院紺＋緑の屋根、複数の塔（Cylinder＋長Cone、先端に魔石Sphere淡光）**、アーチ回廊。大きな校舎＋中庭。
- **看板＋アイコン**：**開いた本＋杖／結晶**（Box本＋細Cylinder杖＋Sphere魔石）。校章プレート。
- **扉**：大アーチ門（学院紋）。
- **窓**：**縦長アーチ窓を多数＋ステンド風の差し色**（学院の華）。塔に丸窓。
- **プロップ**：**中庭（噴水・芝＝緑Box・ベンチ）**、回廊（柱列＋アーチ）、掲示板（時間割・試験・模擬戦の告知＝第三話"新人模擬戦"）、**訓練用の的/結界石（台座on地面、浮かせない）**、学生NPC群（同制服＋個別髪＝そろいだが個人）。
- **区画**：学院区。塔が街のスカイラインに立つ第二のランドマーク。

### ⑭ 夜の歓楽街・情報街 / Night District & Information Alley（雰囲気のみ・非性的）
> 第一話「夜は獣より人が怖い」「酒場が黒帳」。**危うさ・密談・情報**で魅せる。**性的表現は一切作らない。**
- **屋根**：くすみ色＋**密集して傾いた看板群**。狭い空に看板がせり出す＝路地の圧。
- **看板＋アイコン**（**雰囲気記号のみ**）：
  - 酒場サイン（ジョッキ・樽）を密に
  - **踊り子の看板＝シルエットの板**（人型を1枚のBox板で抽象化、扇/リボンを示す曲線Box）＝"演し物がある"の記号に留める（露出・肉体描写なし）
  - **情報屋＝目 or 耳のマーク**（Sphere＝目＋細Box／または封筒Box＝密書）、看板は小さく目立たない
  - 占い/予言の看板（星・月・手のマーク＝LUMEN設定の"予言の点数化"の街版）
- **扉**：暗い入口＋カーテン（布Box板）、覗き窓（小Box）。
- **窓**：**ランタンで縁取られた窓**（橙 emissive を多用）＝路地が灯りで点々と光る＝妖しく美しい。
- **プロップ**：**吊りランタンの列**（ポール/ロープ＋小Cylinder＋黄橙光、`InstancedMesh`で連ねる）＝歓楽街の主役ビジュアル。樽、立ち話のフード付きNPC（情報屋・スリ・監察の影）、路地の狭い段差。**危険の示唆**＝物陰のフードNPC・薄暗がり（明度は落とすが色は残す＝"夜だが絵として綺麗"）。
- **トーン制御**：明るい昼の街と対比して**夜だけ彩度を少し下げ、ランタン橙の点光源で温度を作る**。怖さは"何が起きるか分からない静けさ"（第一話の少年たちの"消えるような去り方"）で出し、グロ・性的描写は出さない。
- **区画**：商業区の裏路地・川沿い・市壁の影。

## B-4. 配置・街割りのガイド（世界観を地図で語る）
- **トライアングル配置**：中央公共区に **冒険者ギルド・大聖堂・役所（記録所）** を三角形に置く＝LUMENの「王権／教会／ギルド」三者均衡を街の幾何で暗示。広場が中心。
- **動線リアリズム**：薬屋はギルド近く（怪我人が来る）、宿はギルドから少し離す（第一話準拠）、倉庫は門・川の物流側、武具通りはまとめる。
- **境界の可視化**：貴族街は**塀＋門＋衛兵**で平民区と分離（通行許可の世界）。スラムは市壁の影に密集。歓楽街は商業区の"裏"へ。
- **ランドマーク2本**：大聖堂の尖塔と魔法学校の塔をスカイラインに立て、プレイヤーが街のどこからでも方角を取れるようにする。
- **明暗の対比**：昼の大通り＝明るく彩度高（Genshin）／夜の路地＝彩度を落としランタン橙で温度。同じ街の二面性＝"明るい絵＋怖い夜"の両立。

## C. 実装チェックリスト / Build Checklist
- [ ] プリミティブ4種（Box/Cylinder/Cone/Sphere）の各形状を**1回だけ生成して共有**。
- [ ] 色は `materialCache[hex]` で `MeshLambertMaterial` を使い回し（Standard不使用）。
- [ ] NPC素体は共通、職業レイヤーを乗せる関数（`addGuildReceptionist(base, seed)` …）。
- [ ] 外見は **index/座標ハッシュで決定論的**に振る（実行ごとに不変）。
- [ ] **頭やや大・大きい瞳＋ハイライト・尖りあご・髪3マス**でアニメ顔を担保。
- [ ] モブは5体に1〜2体だけ**個体差アクセント**（包帯/傷/タグ/荷）で群衆に個を出す。
- [ ] 建物は**接地必須・浮遊パーツ禁止**、各タイプに**看板＋サインアイコン**必須。
- [ ] 量産対象（窓・敷石・屋台・樽・吊ランタン・長屋）は `InstancedMesh`。
- [ ] ライトは Hemisphere＋Directional 1灯、影は主要のみ／夜のみ点光源を少数。
- [ ] 権威の記号（ギルド証・銀印・封蝋・通行札・魔力紋）を**小物として街とNPCに散りばめ**、"鍵取り"でなく"信用と身分"を視覚化。
- [ ] トーン：絵は明るく清潔、制度の痕跡（徽章・台帳・行列・継ぎ当て）を小物で乗せ、夜のみ妖しく（非性的）。
