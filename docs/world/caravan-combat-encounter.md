# Caravan combat encounter polish

Issue #25 の第一段階として、荷車襲撃を「火球1発で終わるイベント」から、短い小戦闘として見せるための軽量な上乗せを追加する。

## 方針

`game_world.js` の既存進行を壊さず、`caravan_combat.js` と `caravan_combat.css` で戦闘UI・HP・予兆・被弾・撃破処理を補助する。

## 追加される体験

- 黒毛の噛み犬に HP 3 を持たせる
- Jキー / LキーでMPを消費して攻撃する
- TARGETを正面に入れて撃つとHPが減る
- 赤い予兆表示後、突進判定が発生する
- 近くにいると被弾し、HPが少し減る
- Kキー回避中は突進を避けやすい
- 撃破時に商人救助・紹介状・北門導線へ進む
- 救助後、警戒混じりの短い反応を表示する

## 制限

- まだ本格的な敵AIではない
- 3D上の敵モデル自体は `game_world.js` の既存beastを使う
- 戦闘の予兆とHPはDOM/CSS中心
- 物理エンジン・経路探索・新アセットは使わない

- 3D空間上の突進移動とDOM予兆を同期する
- 模擬戦イベントにも同じ軽量戦闘基盤を流用する
