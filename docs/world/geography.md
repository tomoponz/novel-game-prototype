# 地理・都市構造設計ノート — 王都アウレリア（Aurelia）

> 対象: 「小説風RPG試作 / 異世界転生RPG 漆黒の翼」Three.js 版（`game_world.js` の `buildCity` マップ）。
> 目的: 既存の3Dマップ座標に**そのまま乗る**地理設定を、開発者が建造物・看板・経路・イベントの配置根拠として使えるようにまとめる。
> トーン方針: **見た目は明るいアニメ/原神調のファンタジー城下町。中身は登録・契約・教会記録・通貨・信用・危険のグラウンデッドな社会**。地理は権力の三角（王権・規範・交換）が石畳の上に固定された配置として読めること。
> 物語軸: 主人公ユウジが**物理的な「鍵」を集めるのではなく、都市の制度の中で信用・身分・場所をじわじわ獲得していく**。地形と区画はその「階段」を可視化する装置として設計する。

---

## 0. 一行で言うと

**丘の上の王城（王権）を頂点に、斜面を住宅が埋め、平地の中央広場から放射する大通りに、大聖堂（規範）と冒険者ギルド（交換）が三角に配置された、明るいアニメ調の丘陵型城郭都市。** ただし路地一本入れば情報と死が転がっている。

LUMENの結論記録（v5.1）の表現を地理に翻訳すると——

- **王城＝強制（王権）／大聖堂＝規範（教会）／ギルド＝交換（市場）** の三項が、物理的に三角形を組んで相互監視している。
- 「奇跡はUI、秩序はOS」: きらびやかな尖塔・噴水・光る窓（UI）の下で、台帳・封蝋・登録札・門税（OS）が都市を実際に動かしている。地理はこの二層を同時に見せる。

## 1. 座標系の確定（エンジン準拠・最重要）

`game_world.js` の `buildCity()` が現行マップの正本。以下は**実装済みの数値**であり、設定はこれに従う（設定がコードを動かすのではなく、設定がコードを説明する）。

| 概念 | エンジン座標 | 物語上の方角 | 備考 |
|---|---|---|---|
| 中央広場の中心（噴水） | `(0, 0)` | 都市の臍（へそ） | `fountain(0,0,4.2)` |
| 広場の像 | `(0, -30)` | 城へ向かう側 | `box(0,4,-30,...)` 初代記念碑（後述） |
| 北門（検問） | `gateDistrict(0, 610)`／門 `z≈665` | **北**（マップ上 +z） | プレイヤーはここから入城し**南（-z）へ**大通りを下って広場へ |
| 王城（丘） | `castleHill()` 基部 `z=-356`、城本体 `z≈-630` | **南＝丘の上手（うわて）** | マップ -z 側。標高が上がる方向 |
| 冒険者ギルド | `(135, -70)` | 広場の**北東〜城寄り** | `guildDistrict` |
| 大聖堂／教会 | `(-185, -85)` | 広場の**北西〜城寄り** | `churchDistrict`、尖塔 `y=32` |
| 市場通り | `(285, 85)` | 広場の**東・門寄り** | `marketDistrict`、屋台60基 |
| 職人区（鍛冶） | `(350, -125)` | **東・城寄り** | `craftDistrict` |
| 貴族街 | `(-250, -350)` | **西・城の麓** | `nobleDistrict`、衛兵巡回 |
| スラム／路地裏 | `(-410, 245)` | **西の隅・門寄り** | `slumDistrict`/`alleyDetails` |
| 外門練習場（入口） | `(380, 330)` | **東・門寄り** | `trainingDistrict` → 別マップ `trainingGround` |
| 宿屋「曲がった匙亭」 | `(92, 82)` | 広場の**すぐ南東** | `locations` の `inn` |

| 環状・格子路 | `0, ±130, ±330, ±430` | 区画割りの碁盤目 | `ROAD_LINES` |

> **方角の約束**: 物語内では「北門」「大通りを南へ」「南に王城」（`plaza_after` の台詞）に従う。
> すなわち **+z = 北（門・外界）／-z = 南＝丘の上手（王城）／+x = 東（市場・職人・練習場）／-x = 西（教会・貴族街・スラム）**。
> 以後、本書は**この物語内方角**で記述する（必要に応じてエンジン座標を併記）。

## 2. 三層の高さ — 丘陵型城郭都市の骨格

アウレリアは平らな円盤ではなく、**北（門）から南（城）へ向かって地面がせり上がる**丘陵都市。`castleHill()` が8段のテラス（`dy=8` ずつ標高+）で斜面を作っている。これを地理として読むと三層になる。

### (A) 王城・城塞（丘の上手 / `z ≈ -630`, 標高 +64m 相当）
- **王城（Aurelia城）**。`castleOnHill` の主塔・四隅塔・尖塔・赤旗。都市のどこからでも見える「天蓋」。
- **王権＝強制**の象徴。徴税・測量・軍役・戴冠の中枢。LUMEN設定では初代司祭王セラフィオンの「鍵儀」記録が**王宮地下庫**に眠る（物語後半の到達点であり、序盤は**登坂ルート未開放＝入れない**＝身分が足りない、という形で「制度の階段」を表現）。
- 城門前は**通行許可（貴族街通行証＋封蝋付き紹介状）**がなければ衛兵に止められる。ユウジが序盤ここへ近づくと門前払い——「物理の壁」ではなく「身分の壁」。

### (B) 斜面市街（テラス住宅群）
- `castleHill()` がテラス上に住宅プリズムを敷き詰める。**段が上に行くほど家がまばらで上等**（`halfW` が縮み、戸数が減る）。
- 物語的含意: **標高＝社会階層**。麓は密集した庶民、上段は富裕。貴族街 `(-250,-350)` はこの斜面の西の取り付き（城の麓の特等席）に置かれている。
- 視覚は原神調の「段々の赤茶屋根」。中身は地代と眺望の格差。

### (C) 平地下町（城下町・`z ≥ -130` 〜 北門）
- 中央広場・市場・職人区・宿・酒場・倉庫・スラムが広がる**生活と取引の層**。第一話の石畳・露店・荷車・蝿の集る路地はすべてこの層の話。
- 主人公の物語の99%はここで進む。**「荷運び・記録・査定補助など非脅威ポジから始める」**（LUMEN OPS/P05）に合致する低層フィールド。

> **建造指針**: 既存の `castleHill` テラスはそのまま「階層の可視化」として使える。追加するなら**段の境目に小門（衛兵詰所）**を置き、上段へ行くほど「ギルド証→教会記録→貴族街通行証」と要求が上がる**ソフトゲート**にする（物理封鎖ではなく、立て札＋衛兵会話＝制度）。

## 3. 権力の三角形 — 配置が権力を語る

LUMENの `TRI-BALANCE`（王権＝強制／教会＝規範／ギルド＝交換）を、広場を重心とする三角形として地表に固定する。

```
                        〔南・丘の上手〕
                        ■ 王城（王権／強制）  (0, -630)
                          ／      ＼
                         ／        ＼
        ✝ 大聖堂(規範)             ⚔ 冒険者ギルド(交換)
        (-185, -85)                 (135, -70)
                ＼                 ／
                 ＼     ◎広場     ／
                  ＼  (噴水 0,0) ／
                   --------------------
                        ↓ 大通り（北へ）
                        ▦ 北門（外界との境界）(0, 665)
```

- **王城は最奥・最高所**: 物理的に「最後にたどり着く場所」。権力は遠く、高く、見えるが触れない。主人公が王権へ近づく＝物語のクライマックス。
- **大聖堂とギルドは広場をはさんで対称**: 教会（西・規範）とギルド（東寄り・交換）が広場で**にらみ合う**配置。第一話でユウジは「教会はそれを嫌う。ギルドはそれを利用する」と言われ、まさにこの二極の間に立たされる。広場＝**二つの権威の係争地**。
- **広場＝三者の監査点**: LUMENは「相互監査の常設化」を説く。広場の噴水・像・掲示板は、三者すべての告知（門税改定・依頼掲示・祈祷会・失踪者情報）が貼られる**公開チャネル**（第一話の掲示板そのもの）。

### 像（`(0,-30)`）の設定
広場の像は**初代・司祭王セラフィオン**の像とする。城（権力の源流）を背に、北門（民の入口）を見下ろす向き。台座には什一税・聖油検査・三符測量の起源が刻まれる——が、LUMEN裏設定では「奇跡はManaの高濃度現象＋基礎術式」。**像はUI、台座の制度文がOS**。主人公の《鑑定眼》で台座を視ると「認証デバイスの注記」が浮かぶ、という後半フックの設置場所に最適。

## 4. 各区画の詳細設定（建造リファレンス）

### 4.1 中央広場（Plaza Central）｜`(0,0)`
- **機能**: 都市の臍。三大路の交点。告知・待ち合わせ・露店の端・公開処分の場。
- **現状要素**: 噴水（治癒の水ではなく**ただの飲料・洗濯・市民の水場**——魔法は高い、を地理で示す）、像、放射状の街灯、囲む高層商店（`centralPlaza` のリング状ファサード）。
- **社会**: 噴水の縁が**情報の中継点**。水汲みの子どもが会話を盗み聞く（第一話「水を運ぶ子どもがやけに丁寧に耳を立てている」）。広場＝合法な情報市場、路地裏＝非合法な情報市場、と対にする。
- **掲示板**: 「冒険者ギルド依頼／門税改定（銀貨にて）／夜間外出禁止（鐘三つ以降）／失踪者情報／祈祷会（未洗礼は参加不可）」。第一話の現物。三権の告知が同居。

### 4.2 大通り（Main Avenue）｜`x=0` 縦軸（幅24）
- 北門 → 広場 → 城へ続く**式典・行軍・凱旋の軸**。石畳（`0x817767`）。
- 物語的には**入城者が最初に歩く道**。両側にアニメ調の家並み（道に正面を向ける `roadFacingAngle`）。明るい第一印象。だが第一話通り、立ち止まると浮く・露店の干し肉・路地の死体は脇道一本で隣接。
- **イベント**: 凱旋・公開戴冠・処刑・祭礼はこの軸で。広場の像→城まで一直線で「見せる」舞台。

### 4.3 市場通り（Market Street）｜`(285, 85)` 東・門寄り
- 屋台60基（`marketDistrict`）。香辛料・干し肉・焼き菓子（第一話の匂い）。
- **門寄りに置く理由**: 隊商・農村からの物資は北門から入り、すぐ東の市場に流れる。**輸入の動線**として門→市場が短い。
- **イベント**: `market` クエスト（薬草束の盗難騒ぎ）。「正義感だけでは商売にならない。証拠と信用がいる」——主人公が**信用**を学ぶ最初の社会実習。
- **通貨**: 銅貨建ての小額決済が回る場所（経済設定: 銅貨≈10円/$1.56、小銅貨≈商いの最小単位）。両替商・秤・真贋印（魔術刻印）の屋台を置くと経済設定が活きる。

### 4.4 職人区（Craftsmen's Quarter）｜`(350, -125)` 東・城寄り
- 鍛冶（`craftDistrict`、炉・金床）。武具・道具・蹄鉄。
- **城寄りに置く理由**: 良質の武具は城・騎士・上級冒険者の需要に近い方が高く売れる。**煙・火・騒音**を住宅密集の広場から離し、かつ城の調達網に近づける合理。
- **社会**: 「登録前の客に刃物は売れない」（`blacksmith`）。**ギルド証＝購買資格**。徒弟三年契約（LAB/MOBILITY設定: 徒弟三年＋移籍制限）の親方が並ぶ。
- 火属性魔法の需要（炉の点火・温度管理）とも結びつく。`ホイミ`＝加温（第一話の検証結果）を「薪を減らす生活魔法」として職人区が買う、という小経済を置ける。

### 4.5 貴族街（Noble District）｜`(-250, -350)` 西・城の麓（門・塀つき）
- 城のテラスの取り付き＝**標高の高い特等地**。`nobleDistrict` は衛兵巡回のみだが、設定上は**門・塀で囲い、通行証がなければ入れない**。
- **ソフトゲート**: 入口に衛兵＋立て札「貴族街通行許可証（封蝋）を提示せよ」。主人公が序盤に近づくと止められる＝**身分の壁の体験**。後半、保証人の署名・封蝋付き紹介状・冒険者ランク上昇で通行許可が下りる。
- LUMEN設定の「聖域特権・租税回避」「格差固定」が住む層。静かで清潔で、明るい外見の裏に既得権。

### 4.6 スラム／路地裏（Slums / Back-Alleys）｜`(-410, 245)` 西の隅・門寄り
- `slumDistrict`＋`alleyDetails`（密集した小屋・木箱・暗い路地・**紫の魔法陣（怪しい取引の痕）**）。`cityFill` でも `x<-300 && z>165` を**スラム屋根**に分岐させ、自然に貧民区の質感を出している。
- **門寄り・西の隅に置く理由**: 城から最も遠く、最も低い谷地。排水と廃棄物が集まり、よそ者と無登録者が溜まる。**情報の地下経済**の本拠。
- **社会**: 「ここでは金より身元が高く売れる」「黒い羽を見たら拾うな。割れた水晶を見ても触るな」（`alley`）。第一話の「川に浮く／牢に入る／消える」噂の発生源。
- **役割**: 広場（合法情報）の裏面。主人公が**正規ルートで詰まったとき**に非正規の情報を買う場所。だが利息（リスク）が高い。LUMEN設定の「黒帳網（諜報・相場・債券）」の末端ノードがここに刺さっている、と後半接続できる。

### 4.7 北門・検問（North Gate / Checkpoint）｜`(0, 610)`、門 `z≈665`
- 都市と外界の境界。`gateDistrict`: 大門＋両脇の塔＋検問所2棟＋衛兵5名。
- **機能**: 入城審査（名・目的・紹介状）、**門税**徴収（「門税改定、銀貨にて」）、夜間封鎖。LUMEN経済設定では**ゲート課金が地域産業保護の価格フロア**＝門税は単なる関所ではなく**経済政策の装置**。
- **イベント**: `north_gate`「名と目的を言え」「紹介状があるならギルドへ」。主人公の**入城＝最初の制度通過**。再入城時は `north_gate_after`（顔パス化＝信用の蓄積を地理で表現）。
- 第一話の「夜は獣より人が怖い」「鐘三つで外に出るな」がここで効く（門の開閉＝鐘と連動、第7章）。

### 4.8 治癒院（Healing House）｜**新規配置提案: 広場とギルドの間、`(70, -40)` 付近**
- 第一話で**ギルドに隣接または内包**して描かれる（ユウジは襲撃後、商会の保証でギルド入口→治癒院へ）。
- **配置案**: 既存ギルド `(135,-70)` の**広場側手前 `(70,-40)`** に独立棟を置き、ギルドと渡り廊下でつなぐ。理由——治癒は危険職（冒険者・護衛・荷運び）の需要が最大であり、ギルドと隣接が合理。
- **社会**: マリウスの作業場。「命に触れてないなら安い。命に触れたなら高い。値段で世界が区切られている」。**未契約者は最低限**（消毒＝薬草、魔法治癒は高額・未契約者はさらに高い）。神聖魔法（治療〜完全治癒、解毒）は**金と身分で段階化**される。氷=銅一枚、痛み止め=銅、といった**値札の地理**。
- LUMEN設定の「誤検挙は即補償（聖貨）」「法廷魔術（鑑識ギルド）」の窓口を後半ここに増設できる。

### 4.9 宿屋「曲がった匙亭」（The Crooked Spoon）｜`(92, 82)` 広場の南東すぐ
- 第一話の宿。女将マルタ。木看板に**曲がった匙の絵**。
- **配置の合理**: 広場・ギルド・市場すべてに徒歩近接。新人冒険者・行商・よそ者が落ち着く一等地だが、貴族街ではないので**手頃**。
- **社会**: 「長逗留ならギルド証か教会の確認書を見せておくれ」（`inn`）。宿泊＝**身分の継続提示**。鐘三つで施錠・窓に板（第一話）。屋根の価値が高い世界。
- **室内化の優先度高**（README「宿屋での夜イベント」）。1階＝食堂兼酒場、2階＝藁の寝台の個室。**情報を耳で拾う最安の場所**（「酒場が黒帳だ」）。

### 4.10 酒場（Tavern）｜**曲がった匙亭1階に内包、または市場寄りに独立 `(150, 60)`**
- 第一話では宿の1階食堂が酒場機能を兼ねる（黒パン＋豆肉煮込み、荒い笑い、賭けの噂、教会が嗅ぎ回る話）。
- **配置案**: MVPは**匙亭1階に統合**（実装コスト低）。拡張時に市場と広場の間 `(150,60)` に独立酒場を置き、冒険者の溜まり場として差別化。
- **役割**: 合法だが半ば闇の情報結節点。「情報は耳から取るのが一番安い。金より安く、血より速い」。模擬戦の賭け、失踪者の噂、誰が誰を保証したか——**人脈と信用の口コミ取引所**。

### 4.11 倉庫（Warehouses）｜**北門と市場の間、`(380, 200)` 付近の帯**
- 隊商物資・市場在庫・ギルド徴収品の保管。
- **配置の合理**: **北門（搬入）→倉庫→市場（販売）** の物流動線を東側に一本通す。門・倉庫・市場を東に集約し、城下町の西（教会・貴族・スラム）と機能を分離。
- **社会**: 倉庫＝**信用取引（手形封蝋制）**の実体。LUMEN設定「手形封蝋制＝現行信用制度の原型」。在庫を担保に商会が手形を切る。主人公が査定補助として在庫検分に雇われる序盤クエスト適地。盗難・横流し・スラムとの裏ルートの舞台にもなる。

### 4.12 魔法学校（学術院）｜**新規配置提案: 城寄り、教会とギルドの中間上手 `(-60, -200)`**
- シナリオ設定で物語中盤の主要舞台（「常識を知るため・知識を得るため学園へ」「学費を毎日バイトで貯めながら通う」「学園で魔神発生」「組織の手」）。LUMEN設定では**学術院＝大賢者アストラが創設、禁呪目録・段位制度の祖**。
- **配置案**: 広場と城の中間、**教会`(-185,-85)`とギルド`(135,-70)`より一段上手（城寄り）の `(-60, -200)`**。理由——
  - 学術院は**知＝準・権力**であり、規範（教会）と交換（ギルド）の上位・城寄りに置くと三角構造に「第4の極」が立体的に乗る。
  - 教会（規範）との緊張（禁呪・神聖魔法の管轄争い）を**隣接**で表現。
  - 学費をバイトで賄う動線上、広場・ギルド・市場へ下りやすい中腹が妥当。
- **建造**: 既存テラス斜面を活かし、**石造の門・中庭・尖塔より低い実用的な塔・魔力測定の上位水晶**。ギルドの測定水晶が「就労資格」なら、学術院の測定は「教育選抜」。主人公の水晶ヒビ（測定不能）が**学術院の関心を引く**フックに使える（シナリオ「荒削りだが才能」→学園へ、に接続）。
- **段位**: 魔法設定の伝位（初伝/中伝/皆伝/極伝）・LV1〜10をここで認定。**推薦状（封蝋付き）**が貴族街通行や上位依頼の信用に化ける。

> **三角→四極のまとめ**: 王城（強制）・大聖堂（規範）・ギルド（交換）・学術院（知）。広場が四者の重心、城が頂点。主人公は広場から出発し、四極の信用を一つずつ積んで斜面を登る。

## 5. 区画配置の論理（なぜそこなのか）

| 軸 | 西（-x） | 中央（x≈0） | 東（+x） |
|---|---|---|---|
| **城寄り（-z, 高所）** | 貴族街・学術院 | 王城・広場の像 | 職人区 |
| **広場帯（z≈0）** | 大聖堂 | 中央広場・宿・酒場 | ギルド・治癒院 |
| **門寄り（+z, 低所）** | スラム・路地裏 | 大通り | 市場・倉庫・練習場 |

読み筋:
- **東（+x）＝生産と交換の側**（市場・職人・倉庫・ギルド・練習場・治癒院）。物が作られ、運ばれ、売られ、人が雇われ、傷を治す「経済の右半身」。
- **西（-x）＝権威と周縁の側**（教会・貴族街・学術院・スラム）。規範・特権・知・そして排除（スラム）が同居する「制度の左半身」。**最上（貴族街）と最下（スラム）が同じ西側に縦に並ぶ**のがこの都市の皮肉——格差は同じ斜面の上下。
- **高さ＝身分**（第2章）。北（門）の谷地ほどよそ者・無登録・貧、南（城）の高所ほど正統・登録済み・富。**主人公は谷から登る**。

## 6. 広域地理 — 農村 → 街道 → 王都門 → 城下町 → 城

現行マップは**王都内（plaza）と森の街道（forestRoad）と外門練習場（trainingGround）**の3フィールド。広域は以下の入れ子で設定する。

```
〔辺境/外縁地帯〕…LUMENの“切り捨てられた外縁”。物価高・治安悪・結界の薄い土地
      │（街道・隊商）
〔農村〕…王都へ食料を供給。隊商の発地。野鼠・害獣の出る畑が城壁外に点在
      │ 街道（forestRoad: 土の街道＋草地＋並木＋岩）
      │   └─ ★ユウジ覚醒地点 / ★隊商襲撃 がここ
〔王都門（北門 z=665）〕…検問・門税・夜間封鎖
      │ 大通り（北→南）
〔城下町（plaza, z≈600→-130）〕…広場・市場・職人・教会・ギルド・宿・スラム
      │ 斜面（castleHillテラス：標高=身分）
〔斜面市街・貴族街・学術院〕
      │ 城門（通行許可制）
〔王城（z≈-630, 丘の頂）〕…王権・地下庫・戴冠
```

### 6.1 ユウジ覚醒地点 / 隊商襲撃 — **城壁の外・森の街道**
- 現行 `buildForestRoad`: マップ中央が街道、`addGate(0,-112)` が王都北門の遠景、`addCaravan(6,15)` が荷車。**ユウジは街道の石畳で目覚める**（第一話「次に目を開けたとき、空があった…石畳。冷たい」——ただし設定上、最初の覚醒は**王都に近い街道・隊商路**とし、ゲームは「街道で目覚め→荷車現場→北門」の順で表現）。
- **隊商襲撃**: 黒毛の噛み犬（`createBeast`）が商人エドリックの荷車を襲う。護衛ガラン。ユウジが石→魔力弾で介入。**城壁の外**で起きるからこそ「治癒は銀貨」「未契約者」の制度がこの後ギルドで突きつけられる。**壁の外＝制度の保護外**。
- 地理的含意: **結界（第7章）と城壁の内側は相対的に安全、外は獣と人の危険**。襲撃が壁外なのは必然。

### 6.2 畑の野鼠クエスト — **城壁の外・農村の畑**
- 第三話の初依頼「街壁の外、畑の鼠。銅三枚」。F級＝雑用＝**城壁外の害獣駆除**。
- **配置**: 北門の外、街道の脇に**農村の畑**フィールドを置く（forestRoad の延長、または専用 `farmland` マップ）。野鼠（猫より大・群れ・病を運ぶ）とボス鼠。
- 物語的含意: **新人は壁の外へ追い出される**。安全な城下町の労働は登録済み・上位者のもの。主人公はまず**最も危険で最も安い壁外労働**から始める（LUMEN「荷運び・記録・査定補助など非脅威ポジ」の最下層）。報酬銅三枚＝経済設定で約30円相当の世界の重み。

### 6.3 外門練習場 — **東門寄り・城壁の内側 `(380,330)`**
- `trainingDistrict` → `trainingGround` マップ。火球試射・翌日の模擬戦。
- **配置の合理**: 「街中で魔法をやられるよりは助かる」（`training_gate`）。**住宅密集の広場から離した東の門寄り**＝魔法の暴発リスクを周縁に隔離する都市計画。第四話のユウジの「明日は外で火を試す。石の上で、水桶と濡れ布を用意して」が制度化された場所。

## 7. 魔法が地理・防衛・流通・時間に与える影響

魔法設定（魔素＝自然界に遍在、器に蓄える／属性／結界・空間・神聖など）を**都市インフラの前提**として地理に落とす。

### 7.1 防衛 — 城壁と結界（二重防御）
- **物理層＝城壁**（`cityWall` ±675、塔、北門）。獣と軍勢を止める石。
- **魔法層＝結界**（結界属性・聖域）。LUMEN設定の**結界都市核（エイジャ炉）**が都市防衛の心臓。アウレリアは大賢者アストラ系の結界都市の系譜とし、**城（または学術院）地下に炉**を置く。結界が獣・瘴気・小規模の禁呪を弱める。
- **地理的帰結**:
  - 壁の**内＝結界圏＝相対的に安全**、**外＝結界の薄い土地＝獣・病・襲撃**（6.1/6.2の事件が壁外で起きる根拠）。
  - 炉の**老朽化＝ブラックアウト＝結界停止**が後半の国家級危機（LUMEN「都市停電危機の修復」「炉の代替運転」）。停電時、**門と広場が避難動線の要**（治癒院と協働した避難経路、LUMENフック第3）。
  - 結界の維持コスト（Mana供給）が**門税・什一税の使途**の一部、という経済接続。

### 7.2 流通・物流 — 門（ゲート）と空間魔法
- **長距離はゲート（門路）**: LUMED設定の**竜脈道（軍用ゲート原型）／門路**。常設長距離ゲートは高価で**国家・大商会・軍**の独占。一般人と物資は**街道＋隊商**が基本（だから6.1の隊商が存在する）。
- **ゲート課金＝価格フロア**: 速さの代価を透明な料金で示す（LUMEN「門：速さの代価は透明な料金で」）。ゲートの発着所を置くなら**北門の外側か城の管理下**（軍・王権の統制物）。
- **空間収納**（空間属性）は希少・高位。一般の荷運びは肉体労働＝主人公の序盤職。**だから倉庫と荷車が要る**。魔法が万能でない＝物流の地理（門・倉庫・市場）が成立する。
- **加温・生活魔法の地産**: `ホイミ`＝加温（第四話）→ 薪の節約。職人区・宿・治癒院に**生活魔法の微小経済**。光（ルーモス）→ 夜の照明（街灯と併存、だが連発不可＝MP）。

### 7.3 治安・時間 — 鐘と門限（カーフュー）
- **鐘＝公的な時計**。第一話「夜間外出禁止、鐘三つ以降」。鐘は**大聖堂（または城）から鳴り、都市全体に響く**（`churchDistrict` 尖塔 or 城）。
- **鐘の段階**:
  - 朝の鐘＝起床・門開放・市場開始（第三話「朝の鐘は夢の中まで刺さる」）。
  - 鐘一つ＝夕刻、注意。鐘二つ＝宿の施錠準備。**鐘三つ＝門限・外出禁止・北門封鎖・宿の施錠（窓に板）**。
- **地理的帰結**:
  - 夜間、**広場と大通りは無人・衛兵巡回のみ、路地裏は逆に活性化**（非合法情報・取引）。「夜は獣より人が怖い」。
  - 門限後に壁外にいると**戻れない＝消える**（第一話/三話の脅し）。主人公の壁外クエスト（畑鼠）は**日没・鐘までに帰還**が制約＝ミニ・タイムプレッシャー設計に使える。
  - 鐘は教会の権威の可聴化（規範が時間を支配）。LUMEN「奇跡はUI」——鐘の音色・儀礼はUI、門の施錠・巡回シフトはOS。

### 7.4 魔法と社会階層の地理
- 神聖魔法（治癒）は**高価×身分で段階化**＝治癒院の「値札の地理」（4.8）。
- 攻撃魔法は**目立つ＝狙われる＝死ぬ**（ガラン/マリウスの公式）。だから**練習場は周縁**、街中詠唱は危険視。魔法の使用許可・登録された魔力紋が後半の制度フック（魔法使用許可証）。
- 測定水晶（ギルド＝就労、学術院＝教育）が**才能の値札化**。主人公の「測定不能・ヒビ」は制度の外＝**管理対象＝注目**。LUMEN「未知は怖がられ、管理され、首輪になる」。

## 8. 物語軸との接続 — 「鍵」ではなく「信用」の地理

> 都市の各所は**物理の関門ではなく、社会的正統性の階段**として機能する。以下は地理に紐づく「信用獲得」ポイント。物理鍵・封印・四鍵しきい値の語は、すべて下記の制度行為に**読み替える**。

| 場所 | 旧来の「鍵」発想（不採用） | **採用する社会的正統性** |
|---|---|---|
| 北門 | 門の鍵を入手 | 紹介状の提示・門税納付・入城審査通過 |
| ギルド `(135,-70)` | ギルドの認証鍵 | **ギルド証**・冒険者ランク・依頼実績 |
| 治癒院 `(70,-40)` | 解錠 | 契約者資格・支払能力・教会確認書 |
| 大聖堂 `(-185,-85)` | 封印を解く | **教会記録**（洗礼・身分記録）・保証人の署名 |
| 貴族街 `(-250,-350)` | 門を破る | **貴族街通行許可証**（封蝋付き） |
| 学術院 `(-60,-200)` | 学院の鍵 | **魔法学校の推薦状**・測定選抜・学費 |
| 倉庫/市場 `(380,200)/(285,85)` | 倉の鍵 | **商会との信用取引**・手形封蝋・査定実績 |
| 王城 `(0,-630)` | 王冠の鍵を奪取 | **多署名の正統性**（王＋教会＋市民代表）／登録された魔力紋／身分保証の積み上げ |

- 主人公の進行＝**北の谷（無登録）から南の頂（正統）への登坂**を、信用の階段で表現。
- LUMEN設定の「鍵」「四鍵しきい値」「黒帳鍵」は、ゲーム前面では**封蝋付き紹介状・署名・推薦状・通行許可・登録記録**として現れる。物理フェッチクエストを主軸にしない。
