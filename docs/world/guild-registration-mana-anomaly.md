# Guild registration and mana anomaly overlay

Issue #27 の第一段階として、ギルド登録〜魔力測定を「王都の制度に捕捉される場面」として見せるための軽量DOM演出を追加する。

## 方針

`game_world.js` の会話・進行フラグ本体には大きく触れず、`guild_anomaly.js` と `guild_anomaly.css` で外付けの記録UIを表示する。

## 追加される体験

- ギルド受付でEキーを押した時、登録照会UIを表示する
- 出身地なし、保証人なし、滞在許可なし、例外承認、F級仮登録を明示する
- 魔力測定でEキーを押した時、測定不能UIを表示する
- 未登録波形、詠唱形式不明、王国式照合なし、ギルドマスター確認を明示する
- 魔力測定時のみ、軽い白紫フラッシュを出す
- ギルドマスター前では、登録者ではなく観察対象として扱われるUIを表示する

## 制限

- 既存会話文そのものはまだ大きく書き換えていない
- 3D水晶のmaterial変更や照明変更はまだ行っていない
- 周囲NPCの実際の行動変化はまだない
- 演出はDOM/CSS中心

## 今後の候補

- `game_world.js` の `DIALOGUES.guild_reception` / `mana_measure` / `guildmaster` の本文を濃くする
- 水晶meshの発光や短いSE再生と連携する
- 記録官NPCや周囲NPCの短い反応を追加する
- 模擬戦後の学院・教会・王権による監視演出へ接続する
