# North gate check overlay

Issue #31 の第一段階として、荷車襲撃後の北門検問を「身分・信用・許可の壁」として見せるための軽量DOM演出を追加する。

## 方針

`game_world.js` の北門会話・進行フラグ本体には触れず、`north_gate_check.js` と `north_gate_check.css` で外付けの検問UIを表示する。

## 追加される体験

- 北門付近でEキーを押した時、紹介状確認UIを表示する
- 商会印、封蝋、保証元、本人身元、入城区分、注意事項を明示する
- 紹介状は信用の仮置きであり、身元証明ではないことを示す
- APPROVED風の軽い封印演出を表示する
- 王都入城を「歓迎」ではなく「条件付き一時通行」として意味づける

## 制限

- 既存の北門会話文そのものはまだ大きく書き換えていない
- `north_gate` / `gate` / `merchant` などの進行フラグは変更しない
- 王都遷移の本体処理は既存のまま
- 演出はDOM/CSS中心

- 門番NPCの視線・姿勢・SEと連携する

- ギルド登録・魔力測定演出と文言を接続する
