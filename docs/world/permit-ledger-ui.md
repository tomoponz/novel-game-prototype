# Permit ledger UI

Issue #33 の第一段階として、紹介状・北門通行・ギルド登録・魔力測定・訓練査定・学院扱いを一覧できる許可証・信用台帳UIを追加する。

## 方針

`game_world.js` の進行フラグ本体には触れず、`permit_ledger.js` と `permit_ledger.css` で外付けの台帳UIを表示する。

## 操作

- `P`: 許可証・信用台帳を開閉
- 背景クリックまたは閉じるボタンで閉じる

## 表示内容

- 現在地
- 身元
- 保証元
- 契約/書類
- 通行
- ギルド登録
- 魔力測定
- 訓練査定
- 学院扱い
- Merchant Trust
- Guild Trust
- Crown Record

## 追加される体験

イベントごとの演出を、プレイヤーがいつでも確認できる社会制度上の状態としてまとめる。

これは強さのステータスではなく、王都アウレリアが主人公をどの制度に仮置きしているかの記録である。

## 制限

- 既存の進行フラグは変更しない
- `game_world.js` は触らない
- 台帳の文言は既存stateからの簡易導出
- まだメニュー内部には統合していない

- ギルド・教会・王権それぞれの記録欄を増やす
