# Training and mock battle foreshadow overlay

Issue #29 の第一段階として、外門練習場〜模擬戦〜学院呼出の導線を「管理下の査定」として見せるための軽量DOM演出を追加する。

## 方針

`game_world.js` の戦闘・進行フラグ本体には触れず、`training_assessment.js` と `training_assessment.css` で外付けの査定UIを表示する。

## 追加される体験

- 外門練習場でEキーを押した時、制御査定UIを表示する
- 火球制御、監督、命中精度、回避、記録先を明示する
- 訓練後〜模擬戦前では、模擬戦が「勝利」ではなく危険度判定であることを明示する
- 学院導線では、戦闘力ではなく未登録魔法体系の観察として意味づける
- 軽いスキャンリング演出で「査定されている」感を出す

## 制限

- 既存の戦闘本体はまだ変えていない
- 模擬戦を本格ボス戦化する処理はまだない
- 3D NPCや訓練場オブジェクトの挙動変更はまだない
- 演出はDOM/CSS中心

## 今後の候補

- `game_world.js` 内部の `training` / `mock_battle` 会話文を濃くする
- 模擬戦を「3回回避 + 1回命中 + 生存」で終わるスクリプト戦闘にする
- `caravan_combat.js` の軽量戦闘基盤を模擬戦へ統合する
- 模擬戦後、学院・教会・王権へ記録共有される演出を入れる
